# Eastern Mystic Ritual UI Assets Skill

This repository only distributes the packaged Codex skill ZIP.

## Download

Download `eastern-mystic-ritual-ui-assets.zip` from this repository, then unzip it into your Codex skills directory.

## Skill Direction

The skill generates art briefs, prompt scaffolds, and production guidance for a vertical mobile game visual direction centered on:

- Lingjian spirit diagnostic devices
- Shanhai ink landscape environments
- Jade, brass, astrolabe, and talisman UI language
- Root-bone result cards
- Ritual buttons and spiritual energy effects

The package includes reference images, style references, asset-type notes, mobile game adaptation rules, and a small prompt scaffold script.
